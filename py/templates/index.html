Pets

Up 
get /animals
searches available animals (searchPets)
By passing in the appropriate options, you can search for available animals in the system

Query parameters

type (optional)
Query Parameter — Return results matching animal type
species (optional)
Query Parameter — Return results matching animal species
breed (optional)
Query Parameter — Return results matching animal breed
size (optional)
Query Parameter — Return results matching animal size
gender (optional)
Query Parameter — Return results matching animal gender
age (optional)
Query Parameter — Return results matching animal age
color (optional)
Query Parameter — Return results matching animal color
coat (optional)
Query Parameter — Return results matching animal coat
status (optional)
Query Parameter — Return results matching adoption status
location (optional)
Query Parameter — Return results by location.
page (optional)
Query Parameter — maximum number of records to return default: 1 format: int32
limit (optional)
Query Parameter — maximum number of records to return default: 20 format: int32

Return type

array[animals]
